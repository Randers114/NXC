In this folder we have two solutions, the first one is the is in 
the Computer_Solution folder which is the solution which has the graph seach
on the computer. It has an exeutable as the first thing in the folder, this
program spawns the C program which runs graph search and runs the bluetooth 
data transfer.

The second folder is the Full Running NXC solution which has the entire program
as NXC code and this program is able to run directly on an NXT brick. 

The Test and Translator folders are source code for programs which were important
in constructing the final program, but they have not been included in the 
final solution.
